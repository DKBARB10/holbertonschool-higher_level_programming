# 5-text_indentantion.py
============================
How to Use 5-text_indentation.py
============================

This module prints a text with 2 lines after each of these characters
 ``print_square(size)``.

Usage
=====

Text is printed with two new lines after each of the characters ``.``, ``?``,
and ``:``.

::

    >>> text_indentantion = __import__('5-text_indentantion').text_indentantion
    >>> text_indentantion("ho")
    ho
